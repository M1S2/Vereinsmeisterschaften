Task<(int, string)> Vereinsmeisterschaften.Core.Contracts.Services.IDocumentService.CreateDocument ( DocumentCreationTypes documentType,
bool createPdf = true )

Create the document indicated by the document type.

For the DocumentCreationTypes.Certificates type, the SetCertificateCreationFilters method must be called before this method to set the filters for the certificate creation. Otherwise the old values are used.

Parameters
documentTypeDocumentCreationTypes used to decide which document type and IDocumentStrategy is used
createPdfTrue to also create a .pdf file
Returns
Task that can be used to run this async. The return parameter is a tuple of (number of created pages in the document, filepath to the last created document (PDF if possible))
